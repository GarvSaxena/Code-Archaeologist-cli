# Code Archaeologist

A Node.js CLI tool for analyzing files and directories.

## Features

- Analyze files and directories
- Display project information
- List files and folders
- Show file sizes and types
- Count files and directories
- Analyze file extensions

## Modules Used

- `fs`
- `path`
- `readline`

## Planned Features

- Total file size
- Largest and smallest file
- Recursive directory scanning
- Line of code counting
- TODO scanner
- Project statistics

## Built With

- Node.js
- JavaScript
